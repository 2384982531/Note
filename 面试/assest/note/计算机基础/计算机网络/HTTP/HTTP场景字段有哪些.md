|       字段       |                             含义                             |
| :--------------: | :----------------------------------------------------------: |
|       Host       |           客户端发送请求时，用来指定服务器的域名。           |
|  Content-Length  | 服务器在返回数据时，会有Content-Length字段，表面此次回应的数据长度。 |
|    Connection    | 常用于客户端要求服务器使用[HTTP长连接]机制，以便其它请求复用。 |
|   Content-Type   |      用于服务器回应时，告诉客户端，本次数据是什么格式。      |
| Content-Encoding |                     说明数据的压缩方法。                     |
