# 命令

> 一般放在/bin		/usr/bin
>
> 
>
> 命令的格式
>
> 命令是一行
>
> 命令的标准格式
>
> 命令名 	-选项名	作用域 	追加作用域 -追加选项
>
>  
>
> 作用域 追加作用域 选项 都可以是多个 空格隔开



- pwd: 显示当前工作目录的绝对路径
- /:根目录
- ~：用户目录
    - Linux系统可以用很多个用户
        - 特殊用户有且只有一个		root
    - 其它通通都是普通用户
- echo：输出
- cat：显示
